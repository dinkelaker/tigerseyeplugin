Introduction

The tiger Eclipse Plug-in is an IDE that strives to support easy creation of EDSLs. This guide describes how to install tiger for development, the currently implemented features and shows on some examples how to create and install a new language. tiger has been previously known as textttPopart, which has been developed by Yevgen Fanshil, Leonid Melnyk, Thorsten Peter, David Marx, Kamil Erhard and Tom Dinkelaker. The used examples are adjusted versions of their prior work.
