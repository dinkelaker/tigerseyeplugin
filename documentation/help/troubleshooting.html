Troubleshooting

Transformation produces weird constructs such as "???"

Weird constructs in translated tigerseye output file. Instead of a special character some construct like "???" appears in the translated file. Possibly the used encoding is not consistent. A solution that always worked for me was to set everything to utf8. Setting the file encoding default in the preferences is the easiest way to do this.

MemoryOutOfBounds PermGenSize (Known issue when using Eclipse 3.4 and 3.6)

A solution that worked for me was to simply increase the PermGenSize as this should now be already the case in the Tigersey_IDE launch configuration
